Visual Studio y Proyecto en C#

Visual Studio

Aspecto Descripción
¿Qué es Visual Studio? Visual Studio es un entorno de desarrollo integrado (IDE) de Microsoft, utilizado para crear aplicaciones en múltiples lenguajes, especialmente C#.
Características Editor inteligente con IntelliSense, depuración avanzada, soporte para .NET y C#, diseño visual de formularios.
Tipos de proyectos Permite desarrollar aplicaciones de consola, escritorio (WinForms/WPF), web (ASP.NET), móviles (Xamarin), juegos (Unity) y más.
Extensiones Visual Studio admite la instalación de extensiones que amplían su funcionalidad, como asistentes de código, temas visuales, herramientas de pruebas y productividad.
Compilación y ejecución Compila el código a lenguaje intermedio y lo ejecuta sobre el CLR (Common Language Runtime), permitiendo la ejecución eficiente y segura de aplicaciones.
Lenguajes compatibles Admite C#, C++, VB.NET, F#, Python, JavaScript, HTML, entre otros mediante extensiones.

Trabajo Grupal en C# - Trimestre 1

Elemento del Proyecto Descripción
Fechas Se solicita ingresar la fecha de ingreso y de salida de productos.
Nombre del producto Nombre personalizado del artículo gestionado en el inventario.
Descripción Detalle del producto o su función dentro del sistema de inventario.
Cantidad Número de unidades gestionadas (entrantes y restantes en bodega).
Valor de compra y venta Precio de adquisición del producto y su valor de comercialización.
Ganancia El programa calcula la ganancia bruta (en dinero) y también en porcentaje.
Ciclo contable El usuario define el ciclo contable donde se enmarca la factura.
Visualización de resultados Se imprime una factura por consola y se muestra una tabla con todos los registros ingresados.

Este trabajo integró programación estructurada, entrada y salida por consola, estructuras de datos, cálculos matemáticos y presentación tabular de información. Fue realizado por Morán, Gonzalez, Guilcapi y Cedeño (3ro N) como parte del segundo trimestre.
